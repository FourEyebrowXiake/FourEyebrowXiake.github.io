基本概念

什么是数据结构

讲到数据结构，一定会提到算法。因为构建数据结构就是为了带来最有效率的算法，去解决特定的问题。

  1. 解决问题方法的效率，跟数据的组织方式有关。
  2. 解决问题方法的效率，跟空间的利用效率有关。（递归：非常占空间）
  3. 解决问题方法的效率，跟算法的巧妙程度有关。下面是一个具体的例子:

（java）

数据结构到底是什么

  • 数据结构是数据对象在计算机中的组织方式：
    • 逻辑结构：有线性结构，树形结构，图（多对多的关系网。
    • 物理存储结构：数组，链表（是连续，还是随机）。
  • 数据对象必定于一系列加载其上的操作相关联。
  • 完成这些操作所用的方法就是算法。

    抽象数据类型

    在java中一个类就是一个数据类型。如果这个类的方法不依赖于具体实现，那它就是抽象的。抽象数据类型只定义是什么，不涉及怎么做。

算法

  • 一个有限指令集
  • 接受一些输入
  • 产生输出
  • 一定在有限步骤之后终止
  • 每一条指令必须
    • 有充分明确的目标
    • 计算机能处理的范围内
    • 描述应不依赖与任何一种计算机以及具体的实现手段

class MaxSubseqSum {
    public MaxSubseqSum(){
    }

    public int MaxResult(int n,int[]list){
        int ThisSum,MaxSum;
        int i;
        ThisSum=MaxSum=0;
        for(i=0;i<n;i++){
            ThisSum+=list[i];
            if(ThisSum>MaxSum){
                MaxSum=ThisSum;
            }
            else if(ThisSum<0){
                ThisSum=0;
            }
        }
        return MaxSum;
    }
